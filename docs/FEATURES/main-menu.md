# Feature: 메인 메뉴 (Main Menu)

> 상위 문서: [CLAUDE.md](../../CLAUDE.md), [PRD.md](../../PRD.md) §5.1

## 개요

시스템 진입 후 표시되는 최상위 화면. 기능별 하위 메뉴로 진입하는 진입점이자, 전체 시료에 대한
요약 정보를 한눈에 보여준다.

## 요구사항

1. 콘솔 시작 시 메인 메뉴를 표시한다.
2. 메인 메뉴는 아래 5개 하위 메뉴로 진입 가능해야 한다.

   | 메뉴 | 설명 | 관련 문서 |
   |---|---|---|
   | 시료관리 | 새로운 시료 등록, 목록 조회, 이름 검색 | [sample-management.md](./sample-management.md) |
   | 주문(접수/승인/거절) | 고객 주문 접수 및 승인·거절 처리 | [sample-order.md](./sample-order.md), [order-approval-rejection.md](./order-approval-rejection.md) |
   | 모니터링 | 상태별 주문 수 및 시료별 재고 현황 확인 | [monitoring.md](./monitoring.md) |
   | 출고처리 | CONFIRMED 상태 주문에 대해 출고 실행 | [shipment.md](./shipment.md) |
   | 생산라인 | 현재 생산 중인 시료 및 대기 중인 생산 큐 확인 | [production-line.md](./production-line.md) |

3. 메인 메뉴에는 전체 시료에 대한 요약 정보(예: 등록된 시료 수, 전체 재고 수량 등)를 표시한다.
   구체적인 표기 항목/수준은 자유롭게 결정한다.
4. 잘못된 메뉴 번호/입력 시 에러 메시지를 표시하고 메인 메뉴로 재진입한다 (프로그램이 종료되면
   안 된다).
5. 종료 메뉴(예: "0. 종료")를 제공해야 한다.

## 입력/출력

- **입력**: 메뉴 번호 (정수)
- **출력**: 하위 메뉴 화면으로 전환, 또는 요약 정보 출력

## 예외 처리

- 숫자가 아닌 입력, 범위를 벗어난 메뉴 번호 → 안내 메시지 후 재입력 요청

## 테스트 관점

- 각 메뉴 번호 선택 시 올바른 하위 컨트롤러/뷰로 라우팅되는지 검증
- 잘못된 입력에 대해 예외 없이 재입력을 요청하는지 검증
- 요약 정보가 실제 저장된 시료 데이터와 일치하는지 검증
